Streamlit Uber App

Introduction

In this lab exercise, you’ll be following along with a tutorial written by the creators of Streamlit. You’ll be working with data from Uber, displaying the data in multiple ways, and creating opportunities for user interaction with the data.

Setup

  1. Head over to the Streamlit Docs.

  2. On the right hand side, there’s a table of contents for this tutorial, you’ll be completing all the sections besides “Share your app” and “Get help”. (If you can’t see the Contents, try zooming out.)

After the tutorial

At the end of this tutorial, you should still have some time in your day to go beyond what’s here. As you work, keep track of things that you want to try when you’re done with the main tutorial! Not only is this lab a way to get familiar with Streamlit, it’s also a good chance to practice your docs-reading skills, which will be very useful in your career.

Instructions

  1. Download this lab’s materials from Frodo (there should just be a blank file for you to work in).

  2. Follow the instructions in Streamlit’s docs, stopping with the “Let’s put it all together” section. See you back here when you’re done.

  3. When you have finished their tutorial, do a little exploration. Perhaps you wondered how to customize the map further. Or maybe you want to expand your page to include more components. You can add in things that you learned from the lecture material. Or just start clicking through the different sections of their docs (except Streamlit Cloud) and attempt to implement the things you’re reading about!

  4. Your lab is done when you’ve added at least 3 features beyond the main tutorial.

When you’re done, upload your code to GitHub.
